AI Desktop Agent: Phase 1
A Python-based personal assistant designed to bridge the gap between AI logic and desktop automation. This is the foundational piece of my journey into AI Agents and Hardware integration.

Features
- Real-time Interaction: Responds to user commands via terminal.
- Web Integration: Quickly launches Google and YouTube searches.
- System Memory: Log-based memory system that saves user notes to a local file.
- Time Intelligence: Fetches and displays real-time system clock data.

Tech Stack
- Language:Python 3.x
- Libraries: `webbrowser`, `datetime` (Standard Libraries)

How to Run
1. Clone the repository: `git clone [YOUR_URL_HERE]`
2. Run the agent: `python project1.py`
